# Mech Style Guide

A comprehensive reference for designing battle mechs, drawing from the greatest mech designs across games, anime, and film.

---

## Overall Silhouette Archetypes

### Heavy Assault
**Inspiration:** BattleTech Atlas, Titanfall Ogre, Pacific Rim Cherno Alpha
- Massive, blocky torso dominates the silhouette
- Wide stance, legs spread for stability
- Shoulders wider than hips
- Armor layered in thick slabs
- Conveys: Unstoppable force, walking tank

### Predator
**Inspiration:** BattleTech Mad Cat, Armored Core, Zone of the Enders Jehuty
- Forward-leaning aggressive stance
- Digitigrade (chicken-walker) legs
- Sharp angles, blade-like protrusions
- Weapon pods integrated into silhouette
- Conveys: Speed, lethality, hunter

### Knight
**Inspiration:** Gundam RX-78, Frame Arms, Virtual On Temjin
- Humanoid proportions but heroic scale
- Defined "face" on head unit
- Shield and sword capability
- Balanced, upright posture
- Conveys: Honor, protagonist energy, versatility

### Industrial
**Inspiration:** Patlabor, MechWarrior Urbanmech, Aliens Power Loader
- Utilitarian, function over form
- Exposed hydraulics and pistons
- Cockpit as obvious cabin
- Tool mounts, work lights
- Conveys: Blue collar, repurposed for war

### Organic
**Inspiration:** Evangelion, Escaflowne, Five Star Stories
- Flowing curves instead of hard edges
- Muscle-like bundles visible
- Joints that flex rather than hinge
- Asymmetry acceptable
- Conveys: Alien, biological horror, grace

## LEGS

### Digitigrade (Chicken Walker)
**The signature mech leg. Used by:** Mad Cat, AT-ST, Armored Core
```
Hip ●━━━━━━━━━━━━━━━━
     \
      \  ← Thigh: SHORT, angled 45-50° forward
       \
        ●━━━ Knee (forward-pointing)
       /
      /   ← Shin: LONG, angled 40-45° backward
     /
    ●━━━━ Ankle
    ├───► Foot (toes forward)
```

**Key principles:**
- Thigh is SHORTER than shin (ratio ~0.7:1.0)
- Knee projects FORWARD of hip
- Creates "lightning bolt" or "Z" shape
- Rotation pivots from JOINTS, not segment centers
- Calf has visible actuator bulge on rear

**Variants:**
| Style | Thigh Angle | Shin Angle | Look |
|-------|-------------|------------|------|
| Aggressive | 50° forward | 50° back | Very compressed, ready to pounce |
| Standard | 45° forward | 40° back | Balanced, versatile |
| Tall | 30° forward | 30° back | More upright, dignified |

### Reverse Joint (Bird)
**Used by:** Some Armored Core builds, experimental designs
- Like digitigrade but MORE extreme angles
- Knee almost horizontal
- Excellent for jumping mechs
- Looks alien/unsettling

### Humanoid (Plantigrade)
**Used by:** Gundam, most super robots
- Standard human leg proportions
- Knee bends backward like human
- More relatable, less intimidating
- Works for "knight" archetype

### Quad/Spider
**Used by:** Spider tanks, Tachikoma
- Four or more legs
- Can be digitigrade or insectoid
- Superior stability
- Alien/drone aesthetic

### Tank Treads
**Used by:** Mobile armors, siege units
- No legs at all
- Treads or hover systems
- Maximum stability and armor
- Limited mobility

## TORSO

### Block Core
**Inspiration:** Atlas, Urbanmech, industrial mechs
```
    ┌─────────────┐
    │  ▓▓▓▓▓▓▓▓▓  │  ← Flat top, vents
    │  █ CORE █   │  ← Single massive block
    │  ▓▓▓▓▓▓▓▓▓  │
    └─────────────┘
```
- Simple rectangular or cubic shape
- Armor is the shape itself
- Cockpit embedded within
- Maximum internal volume

### Split Thorax
**Inspiration:** Mad Cat, many BattleTech designs
```
      ┌───┐
      │ H │         ← Head on top
    ┌─┴───┴─┐
    │ UPPER │       ← Upper torso (weapons)
    ├───────┤
    │ LOWER │       ← Lower torso (twist waist)
    └───┬───┘
        │           ← Narrow waist connection
```
- Upper/lower torso separation allows twist
- Narrower waist creates visual interest
- Upper torso wider for weapon mounts
- More dynamic poses possible

### Swept
**Inspiration:** Armored Core, Zone of the Enders
```
        ╱╲
       ╱  ╲         ← Forward-swept "shoulders"
      ╱ ▓▓ ╲
     ╱ CORE ╲       ← Tapered, aerodynamic
    ╱________╲
```
- Angular, aggressive
- Forward-leaning center of mass
- Implies speed
- Less armor volume, more agility

### Barrel
**Inspiration:** Destroid Tomahawk, some heavy artillery
```
       ┌───┐
      ╱     ╲
     │ █████ │      ← Cylindrical core
     │ █████ │
      ╲     ╱
       └───┘
```
- Cylindrical or oval cross-section
- Good for rotating weapon mounts
- 360° coverage potential
- Feels like a turret

## ARMS

### Weapon Pod
**Inspiration:** Mad Cat, Warhammer, Rifleman
```
    Shoulder ●──┬── Weapon Housing
                │   ╔═══════════╗
            Arm │   ║ ▓▓▓▓▓▓▓▓▓ ║  ← Barrels
                │   ╚═══════════╝
```
- Arm IS the weapon
- No hand/manipulator
- Maximum firepower
- Specialist role

### Humanoid Manipulator
**Inspiration:** Gundam, Patlabor, most anime mechs
```
    Shoulder ●
             │
     Upper ══╪══
             │
     Elbow   ●
             │
    Lower  ══╪══
             │
     Hand   ╔╧╗
            ╚═╝ ← 4-5 fingers
```
- Full hand with fingers
- Can use interchangeable weapons
- Melee capable
- Most versatile

### Claw/Pincer
**Inspiration:** Crabs, industrial mechs
```
    Shoulder ●
             │
         ════╪════
             │
            ╱ ╲
           ╱   ╲    ← Two or three digits
```
- Heavy gripping
- Melee focused
- Intimidating
- Industrial origin feel

### Shield Arm
**Inspiration:** Gundam GM, defensive units
```
    Shoulder ●───────────────┐
                             │
            ╔════════════════╡
            ║   SHIELD       │
            ║   SURFACE      │
            ╚════════════════╡
                             │
                    ●────────┘
```
- One arm dedicated to defense
- May have secondary weapon
- Asymmetric design
- Protector role

## HEAD/COCKPIT

### Sensor Dome
**Inspiration:** Battletech, military vehicles
```
        ╭───╮
       │ ◉◉◉ │     ← Sensor array
        ╰───╯
```
- No "face"
- Purely functional sensor cluster
- Feels like a vehicle, not a being
- Military/utilitarian

### Mono-Eye
**Inspiration:** Zaku, Armored Core, Scopedog
```
       ┌─────┐
       │  ◉  │     ← Single glowing eye/sensor
       └──┬──┘
          │
```
- Single sensor that tracks
- Ominous, predatory
- Enemy/antagonist aesthetic
- Can sweep back and forth

### Visor
**Inspiration:** Gundam, heroic mechs
```
       ┌─────────┐
       │ ▓▓▓▓▓▓▓ │  ← Horizontal visor
       │    V    │  ← Optional "chin" vent
       └────┬────┘
```
- Hero aesthetic
- Implies human-like awareness
- V-fin optional for commander units
- Face-like but abstract

### Cockpit Canopy
**Inspiration:** Patlabor, Titanfall, realistic designs
```
       ╭─────────╮
      ╱   ████   ╲   ← Visible glass/viewport
     │  ┌──────┐  │
     │  │PILOT │  │   ← Pilot visible
     │  └──────┘  │
      ╲__________╱
```
- Pilot visibility emphasizes human element
- More vulnerable feeling
- Relatable, less god-like
- Good for protagonist mechs

### Antenna Array
**Inspiration:** Communications/electronic warfare units
```
          ┃┃┃
         ┃ ┃ ┃
        ┃  ┃  ┃    ← Multiple antenna
       ┌───┴───┐
       │ COMM  │
       └───────┘
```
- No traditional "head"
- Function-specific
- Support unit aesthetic
- Lots of small sensor dots

## SHOULDERS/WEAPON MOUNTS

### Box Launchers
**Inspiration:** Mad Cat, Catapult, missile boats
```
    ╔═══════════╗
    ║ ▫▫▫▫▫▫▫▫▫ ║   ← Missile cells
    ║ ▫▫▫▫▫▫▫▫▫ ║
    ╚═══════════╝
```
- Rectangular boxes on shoulders
- Grid of missile tubes
- Devastating alpha strike
- Signature look for fire support

### Energy Cannons
**Inspiration:** Marauder, PPC boats
```
       ╱══════════════▶
    ●═╬
       ╲══════════════▶
```
- Long barrels extending past shoulder
- Implies power, range
- Often glowing energy vents
- Sniper/artillery aesthetic

### Layered Armor
**Inspiration:** Heavy assault mechs
```
    ┌────────────┐
    │ ╔════════╗ │
    │ ║ ARMOR  ║ │   ← Overlapping plates
    │ ╚════════╝ │
    │  ╔══════╗  │
    │  ╚══════╝  │
    └────────────┘
```
- Shoulders as pure defense
- Layered, overlapping plates
- Protects cockpit from side shots
- Tank aesthetic

### Swept Wings
**Inspiration:** Wing Gundam, flight-capable units
```
         ╱────────────
    ●═══╱
         ╲────────────
```
- Angled panels suggesting flight
- Aggressive, forward-leaning
- May contain thrusters
- Speed/agility implication

## FEET

### Two-Toed (Digitigrade Standard)
**Inspiration:** Mad Cat, most chicken-walkers
```
         │
       ┌─┴─┐      ← Ankle
       │   │
    ╱──┴───┴──╲   ← Two forward toes
       ╲───╱      ← Heel spur
```
- Two prominent forward toes
- Rear heel spur for balance
- Best for digitigrade legs
- Predator/bird aesthetic

### Three-Toed
**Inspiration:** Dinosaurs, alien designs
```
         │
       ┌─┴─┐
    ╱──┴─┬─┴──╲   ← Three even toes
         │
```
- More stability
- Alien/reptilian look
- Good weight distribution
- Slightly slower feel

### Platform
**Inspiration:** Heavy assault, industrial
```
         │
       ┌─┴─┐
    ╔══╧═══╧══╗   ← Flat platform
    ╚═════════╝
```
- Maximum ground contact
- Industrial/utilitarian
- Stable weapon platform
- Less mobility implied

### Clawed
**Inspiration:** Predator mechs, melee units
```
         │
       ┌─┴─┐
      ╱│   │╲
     ╱ └───┘ ╲    ← Sharp digging claws
    ◢       ◣
```
- Aggressive, predatory
- Implies terrain grip
- Melee capability
- Animalistic

## DETAIL ELEMENTS

### Panel Lines
- Break up large flat surfaces
- Suggest access hatches
- Create visual interest
- Should follow structural logic

### Vents/Radiators
- Heat dissipation
- Glowing during combat
- Usually on back, shoulders, legs
- Adds life to static poses

### Pistons/Hydraulics
- Visible mechanical systems
- Industrial aesthetic
- Show joints clearly
- Animate during movement

### Armor Layers
- Overlapping plates
- Creates depth
- Different materials per layer
- Suggests maintainability

### Battle Damage
- Scoring, scratches
- Missing panels
- Exposed internals
- Veteran status

## COLOR SCHEMES

### Military
- Primary: Olive drab, tan, gray
- Secondary: Black, dark brown
- Accent: Red warnings, yellow caution

### Corporate
- Primary: White, light gray
- Secondary: One brand color
- Accent: Black, chrome

### Mercenary
- Primary: Custom, owner's choice
- Secondary: Contrasting
- Accent: Kill markings, personal symbols

### Clan/Faction
- Primary: Faction main color
- Secondary: Rank indicators
- Accent: Unit symbols

### Protagonist
- Primary: White or blue
- Secondary: Red accents
- Accent: Gold/yellow highlights

### Antagonist
- Primary: Dark (black, purple, dark green)
- Secondary: Red or acid green
- Accent: Glowing elements

## ASSEMBLY COMBINATIONS

### "Assault" Configuration
- Torso: Block Core
- Legs: Digitigrade (Standard)
- Arms: Weapon Pod
- Head: Sensor Dome
- Shoulders: Box Launchers
- Feet: Platform

### "Scout" Configuration
- Torso: Swept
- Legs: Digitigrade (Aggressive)
- Arms: Light Manipulator
- Head: Mono-Eye
- Shoulders: Minimal
- Feet: Two-Toed

### "Brawler" Configuration
- Torso: Split Thorax
- Legs: Digitigrade (Standard)
- Arms: Claw + Weapon Pod
- Head: Visor
- Shoulders: Layered Armor
- Feet: Clawed

### "Support" Configuration
- Torso: Barrel
- Legs: Humanoid (stable)
- Arms: Manipulator
- Head: Antenna Array
- Shoulders: Energy Cannons
- Feet: Platform

### "Commander" Configuration
- Torso: Split Thorax
- Legs: Digitigrade (Tall)
- Arms: Humanoid Manipulator
- Head: Visor + V-Fin
- Shoulders: Moderate armor

## IMPLEMENTATION NOTES

### Scale Relationships
```
Light Mech:   scale = 0.7,  bulk = 0.85
Medium Mech:  scale = 0.85, bulk = 1.0
Heavy Mech:   scale = 1.0,  bulk = 1.1
Assault Mech: scale = 1.1,  bulk = 1.25
```

### Joint Hierarchy
```
Pelvis (physics root)
├── Torso
│   ├── Head
│   ├── Left Arm → Shoulder → Upper → Elbow → Lower → Hand
│   ├── Right Arm → ...
│   └── Weapon Mounts
├── Left Leg → Hip → Thigh → Knee → Shin → Ankle → Foot
└── Right Leg → ...
```

### Rotation Pivots
- ALL rotations happen at JOINTS (top of segment)
- Use `setPivotPoint()` to move pivot from center to edge
- Thigh rotates from hip
- Shin rotates from knee
- Never rotate from segment center

### Material Zones
1. **Primary** - Main armor plates (team color)
2. **Secondary** - Accent armor, trim (complementary)
3. **Dark** - Joints, recesses, frame (near-black)
4. **Metal** - Actuators, pistons, chrome (metallic)
5. **Glow** - Vents, eyes, heat sinks (emissive)
